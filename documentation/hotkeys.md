# Hotkeys

F1 - Print Position, Rotation, and Heading in Console
F2 - Player List / Leaderboard

### Chat View

-   Up - Last Message
-   Down - Next Message
-   Right - Autocomplete Command
-   Escape - Exit Chat
-   Enter - Send Chat Message
-   T - Toggle Chat Box

### No Clip

-   Scroll Up - Slow Down
-   Scroll Down - Speed Up
-   WASD - Move Camera
